# TODO Tasks

This directory contains the persistent task system used by all AI agents.

Each task is represented by a single markdown file:

```text
TASK-XXX-description.md
```

Example:

```text
TASK-051-navbar-livewire-fix.md
```

The task file is the source of truth.

Tasks support:
- multi-agent collaboration
- handoffs
- locking
- reviews
- testing
- auditability

Read before working:
- AGENTS.md
- CLAUDE.md
