# Terminal Tooling

## rg

Prefer `rg` over grep.

Use for:
- routes
- policies
- tenant logic
- transactions
- Livewire components

## bat

Prefer `bat` over `cat`.

Use for:
- logs
- PHP files
- Blade templates
- migrations

## fzf

Use for:
- fuzzy search
- navigation
- project exploration

## lazygit

Use for:
- branch management
- commit review
- merge inspection
