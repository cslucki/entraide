# T074.1 — Assets UX ChatLoop

Ces images sont des **mockups conceptuels** générés pour cadrer l'UX de ChatLoop sur mobile, desktop et administration.

Elles **ne sont pas** :

- des screenshots Playwright de validation
- une UI finale pixel-perfect
- des specs CSS/Tailwind définitives

Elles servent à :

- guider la spécification produit T074.2
- aligner les parties prenantes sur les parcours UX
- illustrer la séparation produit Boucles / Échanges / Objectifs / Actus

Ne pas les ranger dans `ai/playwright/screenshots/`.  
Ce dossier reste réservé aux vrais screenshots de validation Playwright.
