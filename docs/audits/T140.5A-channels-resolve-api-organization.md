# Audit T140.5A — Channels + ResolveApiOrganization

## Objectif

Basculer les broadcast channels et le middleware API en organization_id-first,
avec community_id fallback documenté.

## Périmètre

- `routes/channels.php` — comparaison tenant sur le channel privé `loop.{loopId}`
- `app/Http/Middleware/ResolveApiOrganization.php` — resolution organisation pour les routes API
- `tests/Feature/T1405ARuntimeOrganizationIdTest.php` — tests dédiés

## Changements

### routes/channels.php (ligne 25)

**Avant :**
```php
if ($loop->community_id !== $user->community_id) {
```

**Après :**
```php
$orgId = $user->organization_id ?? $user->community_id;
if ($loop->organization_id !== $orgId) {
```

**Fallback :** `$user->community_id` si `organization_id` est null.
**Colonne :** `$loop->organization_id` (préférée) au lieu de `community_id`.

### ResolveApiOrganization.php — resolveFromAuthenticatedUser

**Avant :**
```php
if (! $user->community_id) {
    return null;
}
return Organization::whereKey($user->community_id)
    ->where('is_active', true)
    ->first();
```

**Après :**
```php
$orgId = $user->organization_id ?? $user->community_id;
if (! $orgId) {
    return null;
}
return Organization::whereKey($orgId)
    ->where('is_active', true)
    ->first();
```

**Fallback :** `$user->community_id` si `organization_id` est null.

### ResolveApiOrganization.php — bindOrganization

**Avant :**
```php
app()->instance('current_organization', $organization);
```

**Après :**
```php
app()->instance('current_organization', $organization);
if (! app()->bound('current_community')) {
    app()->instance('current_community', $organization);
}
```

**Alignement :** Même comportement que `ResolveUrlOrganization::bindOrganization()`.

## Risques résiduels

1. **community_id DB toujours présent** — colonne legacy non supprimée. Les créations via `HasOrganizationId` sync toujours `community_id`. Acceptable.
2. **current_community fallback maintenu** — le bind `current_community` est conservé pour compatibilité avec le code legacy qui y accède. Non supprimé.
3. **Routes legacy /{community} inchangées** — pas de redirect, pas de SEO impact.
4. **BelongsToTenantScope unchanged** — filtre toujours sur `organization_id` comme avant.
